About CSS

CSS (Cascading Style Sheets) is a rules-based programming language developed by Håkon Wium Lie, the World Wide Web Consortium (W3C) in 1996 for a simple reason. The HyperText Markup Language (HTML) that is used to format and link to content on web pages was not intended to allow for advanced styling or layout capability. CSS is not technically a necessity, but a web page that features only HTML elements would look very plain and out of date. It is like a decoration system for your very plain webpage.
